1 - Centering Vector

Referencing this Youtube Video, there is no tools available at the moment that can help us to automatically center an object like in inkscape. One trick is to use guides, to do that, simply

  1. Select the Background Canvas, unlocked it and press ctrl + T, drag the rulers to the center of the canvas as shown in image below.

  2. Next, select the vector canvas, and using the select shapes tool, click on the red box and drag it to the middle of the canvas.

That’s it!
